# Kesimpulan
Menurut saya, Enkapsulasi (encapsulation) adalah sebuah metode untuk mengatur struktur class dengan cara menyembunyikan alur kerja dari class tersebut. Dengan enkapsulasi, kita bisa memilih property dan method apa saja yang boleh diakses, dan mana yang tidak boleh diakses. Dengan menghalangi kode program lain untuk mengubah property tertentu, class menjadi lebih terintegrasi, dan menghindari kesalahan ketika seseorang ‘mencoba’ mengubahnya. Untuk membatasi hak akses kepada property dan method di dalam sebuah class, terdapat 3 kata kunci, yakni Public, Protected dan Private. Kata kunci ini diletakkan sebelum nama property atau sebelum nama method. Dalam Java, enkapsulasi dapat dilakukan dengan pembentukan kelas-kelas menggunakan keyword class. Disini kita dapat membuat class baru yang dibangun di atas class yang ada.
